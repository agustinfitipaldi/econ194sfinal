State Tables

The State Banking Performance Summary provides key financial and ratio data for FDIC-insured institutions. View data by all institutions, commercial banks (national charters and state charters) and savings institutions (savings banks, federal charters, and state charters) in each state. Data tables are available dating back to the March 31, 1995, reporting period. Please see the Data Sources, Computation Methodology, and Accounting Changes sections of the Notes to Users for more details.
